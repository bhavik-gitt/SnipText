# SnipText

SnipText is a lightweight React web app for fast text transformations and quick readability insights. It helps users format, clean, and analyze text directly in the browser.

## Features

- Convert text to **UPPERCASE** or **lowercase**
- Remove extra spaces from text
- Copy processed text to clipboard
- Download text as a `.txt` file
- Live word count and character count
- Estimated reading time calculation
- Light/Dark mode toggle
- About page with project overview

## Tech Stack

- React
- React Router
- Bootstrap (utility and component styling)

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm

### Installation

```bash
git clone https://github.com/bhavik-gitt/SnipText.git
cd SnipText
npm install
```

### Run Locally

```bash
npm start
```

The app will run at: `http://localhost:3000`

## Available Scripts

In the project directory, you can run:

### `npm start`
Starts the development server.

### `npm test`
Runs the test suite in watch mode.

### `npm run build`
Builds the app for production in the `build/` directory.

### `npm run eject`
Ejects the Create React App configuration (irreversible).
